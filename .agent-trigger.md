# IDE Agent Trigger File
File ini secara otomatis digenerate dan dimodifikasi oleh Web Dashboard untuk berkomunikasi dengan IDE Agent (Trae/Kiro/Cursor).

**Instruksi untuk IDE Agent:**
1. Pantau file ini. Jika ada instruksi baru di bawah ini, segera eksekusi tanpa perlu konfirmasi manual jika aman.
2. Hapus baris yang sudah dieksekusi, atau tandai dengan `[DONE]`.
